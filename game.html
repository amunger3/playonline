This online game is assisted by a calculator which will give you hints to find the optimal strategy!
Seveval game modes are possible, from 3 (very easy) to 7 columns (expert) and from 6 to 10 colors
At each stage of the game, the number of remaining possible codes is automatically computed and displayed.
The calculator also compares what you played to the "optimal mathematical strategy". For each code played,
it compares "the average number of attempts to find secret codes" which is reachable when an optimal code is played
to this average number when your code is played, and then displays the difference next to your code.
For example:
  • +0.00 means that what you played was optimal (no difference between your code and the optimal codes),
  • -1.00 means that what you played was totally useless (1 attempt was lost, thus -1),
  • -0.50 means that what you played was not optimal: you lost 1/2 attempt on average
    to reach secret codes in comparison to an optimal code. That means that if the optimal strategy reaches
    the secret code in 4 attempts on average, your code (followed by an optimal strategy) will reach it in 4.5 attempts on average.
    Of course, if you are lucky, this inefficient code may be the secret code, in which case you will win instantly the game! :)

Summary of the rules:
The goal of the game is to find out a secret code composed of N colors (duplicate colors being allowed, N being between 3 and 7).
The player makes successive attempts and has to find it out before maximum number of attempts is reached.
At each attempt, the program will provide feedback by displaying from zero to N pegs next to the code played:
  • a black peg indicates the existence of a correct color placed in a correct position,
  • a white peg indicates the existence of a correct color placed in a wrong position.
Once feedback is provided, another attempt is made.
The game ends when the secret code is found (which means the player got N black pegs) or when the maximum number of attempts is reached
meaning the game is lost.
